## 链表的概念

​       链表存储有序的元素集合，但不同于数组，链表中的元素在内存中并不是连续放置，和数组相比，链表的好处在于，添加或移除元素的时候不需要移动其他元素，劣势在于访问链表中间的一个元素时，需要从起点开始迭代链表直到找到所需元素，而数组可直接访问。

## 单向链表

![](https://s1.ax1x.com/2020/04/23/JdmDIK.png)

## 单向链表-特点

1. 用一组任意的内存空间去存储数据元素（内存可以是连续的，也可以是不连续的 ）
2. 每个节点都由数据本身和一个指向后续节点的指针组成
3. 整个链表的存取必须从头指针开始，头指针指向第一个节点
4. 最后一个节点的指针指向null

## 单向链表-创建

```
class Node {
  constructor(element) {
    this.element = element
    this.next = null
  }
}

class LinkList {
  constructor() {
    this.head = null
    this.length = 0
  }
  // 追加元素
  append(element) { ... }
  // 任意位置插入元素
  insert(position, element) { ... }
  // 移除指定位置元素
  removeAt(position) { ... }
  // 寻找元素下标
  findIndex(element) { ... }
  // 删除指定文档
  remove(element) { ... }
  // 输出字符串
  toString() { ... }
}

const list = new LinkList()
```

## 单向链表-追加元素

```
  // 追加元素
  append(element) {
    let node = new Node(element)
    let current = null
    if (this.head === null) {
      this.head = node
    } else {
      current = this.head
      while (current.next) {
        current = current.next
      }
      current.next = node
    }
    this.length++
  }
```

## 单向链表-任意位置插入元素

```
  // 任意位置插入元素
  insert(position, element) {
    if (position >= 0 && position <= this.length) {
      let node = new Node(element)
      let current = this.head
      let previous = null
      let index = 0
      if (position == 0) {
        this.head = node
      } else {
        while (index < position) {
          previous = current
          current = current.next
          index++
        }
        node.next = current
        previous.next = node
      }
      this.length++
      return true
    }
    return false
  }
```

## 单向链表-移除指定位置元素

```
  // 移除指定位置元素
  removeAt(position) {
    if (position >= 0 && position < this.length) {
      let index = 0
      let current = this.head
      let previous = null
      if (position == 0) {
        this.head = current.next
      } else {
        while (index < position) {
          previous = current
          current = current.next
          index++
        }
        previous.next = current.next
      }
      this.length--
      return current.element
    }
    return null
  }
```

## 单向链表-寻找元素下标

```
  // 寻找元素下标
  findIndex(element) {
    let current = this.head
    let index = 0
    if (current) {
      while (current.element !== element) {
        index++
        current = current.next
      }
      return index
    }
    return -1
  }
```

## 单向链表-删除指定文档

```
  // 删除指定文档
  remove(element) {
    let index = this.findIndex(element)
    return removeAt(index)
  }
```

## 单向链表-输出字符串

```
  // 输出字符串
  toString() {
    let current = this.head
    let str = ""
    while (current) {
      str += `${current.element}`
      current = current.next
    }
    return str
  }
```

## 双向链表

![](https://s1.ax1x.com/2020/04/23/JdIXTS.png)

## 双向链表-特点

1. 多了一个指向上一个节点的指针

## 双向链表-双向链表的创建

```
class Node {
  constructor(element) {
    this.element = element
    this.prev = null
    this.next = null
  }
}

class DoubleLinkList{
  constructor(){
    this.head = null // 头
    this.tail = null // 尾
    this.length = 0
  }
}

const doubleList = new DoubleLinkList()
```

## 双向链表-任意位置插入元素

```
  insert(position, element) {
    if (position >= 0 && position <= this.length) {
      let node = new Node(element)
      let index = 0
      let current = this.head
      let previous = null
      if (position === 0) {
        if (!this.head) {
          this.head = node
          this.tail = node
        } else {
          node.next = current
          this.head = node
          current.prev = node
        }
      } else if (position === this.length) {
        current = this.tail
        current.next = node
        node.prev = current
        this.tail = node
      } else {
        while (index < position) {
          previous = current
          current = current.next
          index++
        }
        node.next = current
        previous.next = node
        current.prev = node
        node.prev = previous
      }
      this.length++
      return true
    }
    return null
  }
```
